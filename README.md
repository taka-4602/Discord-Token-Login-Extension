# Discord Token Login Extension

Discord上でトークンを使用してログインするためのChrome拡張機能です。  
複数アカウントの管理、一括インポート、メモ機能、高度なアカウント管理機能を提供します。

## 機能

### 基本機能
- **トークンログイン**: Discordトークンを使用した迅速なログイン
- **アカウント保存**: トークンとプロファイル情報の安全な保存
- **複数アカウント管理**: 保存したアカウントの簡単な切り替え
- **クリップボード統合**: ワンクリックでトークンをペーストしてログイン

### 一括管理機能
- **一括インポート**: テキストファイルまたは手動入力でトークンを一括追加
  - サポート形式: `token/token`, `token token`, `token,token`
  - 重複検出とスキップ機能
- **ソート機能**: 7つの基準でアカウントリストをソート
  - 最新/最古、名前昇順/降順、インポート順、エラー順
- **エクスポート機能**: 全トークンまたは有効なトークンのみをテキストファイルで出力

### 高度な機能
- **メモ機能**: 各アカウントにメモを追加して管理
- **トークン編集**: メモモーダル内でトークンを直接編集
- **トークンコピー**: ワンクリックでトークンをクリップボードにコピー
- **トークンテスト**: トークンの有効性をリアルタイムで確認
- **ステータス表示**: インポート済み、ログイン失敗の視覚的表示
- **スクロール対応**: 大量のアカウントリストでも快適に操作

### インターフェース
- **ポップアップ版**: 軽量で素早いアクセス用（320x600px）
- **タブ版**: 高機能版（拡張可能、フルスクリーン対応）
  - ポップアップから「Open in Tab」ボタンで切り替え
  - より広いワークスペースでの快適な操作

## インストールと使用方法

### Chrome拡張機能として使用

1. リポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions/` を開く
3. 「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `chrome_extension` フォルダを選択
6. 拡張機能アイコンをクリックしてポップアップを開く

### 機能の使い方

#### 基本的なログイン
1. Discordトークンを入力欄に貼り付け
2. 「Save Account」をオンにして保存（オプション）
3. 「Login to Discord」をクリック

#### 一括インポート
1. 「Bulk Import Tokens」セクションを展開
2. テキストファイルをアップロードするか、テキストエリアに直接入力
3. 「Process Tokens」をクリック
4. 重複チェック後、新規トークンが自動保存される

#### アカウント管理
- **メモ追加**: アカウント横の📝アイコンをクリック
- **ソート**: ドロップダウンメニューから希望の並び順を選択
- **削除**: アカウント横の×ボタンをクリック
- **全削除**: 「Clear All」を2回クリック

#### タブ版の使用
1. ポップアップで「Open in Tab」をクリック
2. 新しいタブでフル機能版が開く
3. エクスポート/インポート、高度なトークン管理が可能

## ファイル構成

```
chrome_extension/
├── manifest.json       # 拡張機能の設定
├── index.html         # ポップアップ版UI
├── tab.html          # タブ版UI（高機能）
├── script.js         # ポップアップ版ロジック
├── tab-script.js     # タブ版ロジック
├── style.css         # 共通スタイル
├── content.js        # Discord上での動作制御
└── icon_*.png        # アイコン画像
```

## 開発・テスト

### ローカルテスト
```bash
# test.htmlを開いてローカルテスト
open test.html
```

### デバッグ
- Chrome DevToolsでポップアップをデバッグ: F12 → Consoleタブ
- タブ版のデバッグ: 通常のWebページと同様にF12
- ストレージ確認: DevTools → Application → Storage → Extension

## セキュリティ

- トークンはローカルストレージにのみ保存
- 外部サーバーへの送信は一切なし
- Discordの公式APIのみ使用
- すべての通信はHTTPS
